Stroke-Prediction-Web-Application








A Python-based web application designed to identify patients at risk of stroke using machine learning techniques.

<h3>Objective</h3> 
<ul> 
    <li>To develop a Python-based web application for early identification of stroke risk in patients.</li> 
    <li>To predict the likelihood of a stroke based on input parameters such as gender, age, existing health conditions, and smoking status.</li> 
    <li>To assist healthcare practitioners and individuals in making informed health decisions through predictive analytics.</li> 
</ul>

<h3>Technologies Used :</h3> 
<ol> 
    <li>Supervised Machine Learning (Classification Models)</li> 
    <li>Flask (Backend Web Framework)</li> 
    <li>HTML, CSS, Bootstrap & JavaScript (Frontend Development)</li> 
</ol>
<h3>Installation</h3> 
<ol> 
    <li>Ensure you have Python installed (Python 3.x recommended).</li> 
    <li>Clone the repository: <pre><code>git clone https://github.com/your-username/stroke-prediction-web-app.git</code></pre> </li> 
    <li>Navigate into the project directory: <pre><code>cd stroke-prediction-web-app</code></pre> </li> 
    <li>Create a virtual environment: <pre><code>python -m venv venv</code></pre> </li> 
    <li>Activate the virtual environment: <pre><code>venv\Scripts\activate (Windows) source venv/bin/activate (Linux/Mac)</code></pre> </li> 
    <li>Install required dependencies: <pre><code>pip install -r requirements.txt</code></pre> </li> 
</ol>
<h3>Deployment Instructions</h3> 
<ol> 
    <li>Run the Flask application: <pre><code>python app.py</code></pre> </li> 
    <li>Open your browser and navigate to: <pre><code>http://127.0.0.1:5000/</code></pre> </li> 
</ol>
